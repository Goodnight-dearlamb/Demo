# typescript

## 数据类型

+ 布尔类型（Boolean）
``` js
let flag: boolean = true;
flag = false;
```

+ 数字类型（number）
``` js
let a: number = 1;
a = 2;
```

+ 字符串类型 （string）
``` js
let str: string = "字符串";
str = `字符串的${a}`;
```

+ 数组类型 （array）
  + 第一种方式
  ``` js
  let arr: number[] = [1, 2, 3];
  ```
  + 第二种方式
  ``` js
  let arr2: Array<string> = ["1", "2", "3"];
  ```
